## 👤 About

This is the personal portfolio of **Mohammad Fozle Rabbi**, a Professional Graphic Designer based in Narayanganj, Dhaka, Bangladesh — with **5+ years of experience** specializing in visual identities, logo design, business cards, letterheads, and marketing materials.

---

## ✨ Features

- 🌑 **Dark Cinematic Theme** — Deep black background with electric cyan accents
- 🖱️ **Custom Animated Cursor** — Ring + dot cursor with hover reactions
- 🎞️ **Scroll Reveal Animations** — Staggered fade-in effects on every section
- 📊 **Animated Skill Bars** — Progress bars triggered on scroll
- 🔤 **Editorial Typography** — Bebas Neue + Cormorant Garamond font pairing
- 📜 **Scrolling Marquee Strip** — Showcasing design specialties
- 📱 **Fully Responsive** — Optimized for mobile, tablet, and desktop
- 🖼️ **Embedded Images** — Self-contained, no external image hosting needed
